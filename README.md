# Priyavar – Men's Ethnic Wear Website

[![Live Site](https://img.shields.io/badge/Live%20Website-Visit%20Now-8B0000?style=for-the-badge&logo=wordpress&logoColor=white)](https://priyavar.in/)

## 📋 About the Project

**Priyavar** is a premium men's ethnic clothing brand based in **Jaipur**.  
The website showcases and sells high-quality ethnic & Indo-Western outfits for weddings, festivals, receptions and special occasions with a strong focus on comfort, perfect fit, and modern-traditional fusion.

### ✨ Highlights
- 500+ premium ethnic designs
- 10+ years of fashion expertise
- 70% repeat customers
- Custom tailoring & 30-day exchange
- “Luxury is not loud” philosophy

## 🚀 Local Setup Instructions

1. Clone this repository
2. Import `database/priyavar_2025.sql`
3. Update `wp-config.php` with local DB credentials
4. Copy `wp-content` folder
5. Run on LocalWP / XAMPP / Laragon
6. Visit `http://localhost/priyavar` and login
